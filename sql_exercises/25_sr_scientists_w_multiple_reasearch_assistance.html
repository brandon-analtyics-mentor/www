Senior Scientists with Multiple Research Assistants

Problem Description

Table: Researcher

+---------------+------+
| Column Name   | Type |
+---------------+------+
| researcher_id | int  |
| researcher_name | varchar |
| research_area | varchar |
| supervisor_id | int  |
+---------------+------+

researcher_id is the primary key (column with unique values) for this table.
Each row of this table indicates the name of a researcher, their research area, and the id of their supervisor.
If supervisor_id is null, then the researcher does not have a supervisor (they are a senior scientist).
No researcher will be the supervisor of themselves.

Write a solution to find senior scientists who supervise at least four research assistants.

Return the result table in any order.

The result format is in the following example.

Example

Input:
Researcher table:

+---------------+---------------+---------------+---------------+
| researcher_id | researcher_name | research_area | supervisor_id |
+---------------+---------------+---------------+---------------+
| 201           | Maria         | Genetics      | null          |
| 202           | Alex          | Genetics      | 201           |
| 203           | Sarah         | Genetics      | 201           |
| 204           | Thomas        | Genetics      | 201           |
| 205           | Lisa          | Genetics      | 201           |
| 206           | Robert        | Microbiology  | 208           |
| 207           | David         | Genetics      | 201           |
| 208           | Elena         | Microbiology  | null          |
| 209           | Michael       | Microbiology  | 208           |
| 210           | Carol         | Microbiology  | 208           |
+---------------+---------------+---------------+---------------+

Output:

+---------------+
| researcher_name |
+---------------+
| Maria         |
+---------------+

Explanation:

  • Maria (researcher_id = 201) supervises 5 research assistants: Alex, Sarah, Thomas, Lisa, and David.
  • Elena (researcher_id = 208) supervises 3 research assistants: Robert, Michael, and Carol. Since this is less than 4, Elena is not included in the result.
